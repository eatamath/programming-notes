### Sigma Algebra

Let $X$ be some set, and $\mathcal{P}(X)$ be the power set of $X$. Then a subset $\Sigma \in \mathcal{P}(X)$ is called a sigma algebra if it satisfies:

1. $X \in \Sigma$
2. closed under completion: if $A \in \Sigma$, then $X/A \in \Sigma$
3. closed under countable union: if $A_1,A_2,..,A_n \in \Sigma$, then $\bigcup_{i}^{\infty}{A_i} \in \Sigma$

[reference]([https://en.wikipedia.org/wiki/%CE%A3-algebra#Sub_%CF%83-algebras](https://en.wikipedia.org/wiki/Σ-algebra#Sub_σ-algebras))

### Probability Space

A probability space consists of three parts:

1. A [sample space](https://en.wikipedia.org/wiki/Sample_space),$\Omega$, which is the set of all possible outcomes.
2. A set of [events](https://en.wikipedia.org/wiki/Event_(probability_theory)) $\mathcal{F}$, where each event is a set containing zero or more [outcomes](https://en.wikipedia.org/wiki/Outcome_(probability)).
3. The assignment of [probabilities](https://en.wikipedia.org/wiki/Probability) to the events; that is, a function $P$ from events to probabilities.

Let $(X, \Sigma)$ and $(Y, T)$ be measurable spaces, meaning that $X$ and $Y$ are sets equipped with respective $\sigma$ -algebras $\Sigma$ and $T .$ A function
$f: X \rightarrow Y$ is said to be measurable if for every $E \in T$ the pre-image of $E$ under $f$ is in $\Sigma$; i.e.
$$
f^{-1}(E):=\{x \in X | f(x) \in E\} \in \Sigma, \quad \forall E \in \mathrm{T}
$$

If $f: X \rightarrow Y$ is a measurable function, we will write
$$

$$

#### Conditional Expectation

Exercise 2. Suppose that the $\sigma$ -algebra $\mathscr{G}$ is finite, that is, suppose that there is a finite measurable partition $B_{1}, B_{2}, \ldots, B_{n}$ of $\Omega$ such that $\mathscr{G}$ is the $\sigma$ -algebra generated by the sets $B_{i}$. Show that for any $X \in L^{2}(\Omega, \mathscr{F}, P)$

$$
E(X | \mathscr{G})=\sum_{i=1}^{n} \frac{E\left(X \mathbf{1}_{B_{i}}\right)}{P\left(B_{i}\right)} \mathbf{1}_{B_{i}} \quad \text { a.s. }
$$
Tower Property: If $\mathscr{H}$ is a $\sigma$ -algebra contained in $\mathscr{G},$ so that $\mathscr{H} \subseteq \mathscr{G} \subseteq \mathscr{F},$ then

$$
E(X | \mathcal{H})=E(E(X | \mathscr{G}) | \mathcal{H})
$$

#### Codomain

##### definition

In [mathematics](https://en.wikipedia.org/wiki/Mathematics), the **codomain** or **target set** of a [function](https://en.wikipedia.org/wiki/Function_(mathematics)) is the [set](https://en.wikipedia.org/wiki/Set_(mathematics)) *Y* into which all of the [output](https://en.wikipedia.org/wiki/Output_(mathematics)) of the function is constrained to fall. #### Abelian Group

- Closure
  - For all *a*, *b* in *A*, the result of the operation *a* • *b* is also in *A*.
- Associativity
  - For all *a*, *b*, and *c* in *A*, the equation (*a* • *b*) • *c* = *a* • (*b* • *c*) holds.
- Identity element
  - There exists an element *e* in *A*, such that for all elements *a* in *A*, the equation *e* • *a* = *a* • *e* = *a* holds.
- Inverse element
  - For each *a* in *A*, there exists an element *b* in *A* such that *a* • *b* = *b* • *a* = *e*, where *e* is the identity element.
- Commutativity
  - For all *a*, *b* in *A*, *a* • *b* = *b* • *a*.
